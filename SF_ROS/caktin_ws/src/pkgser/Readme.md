src（include） 文件
	源代码，机器人硬件的底层控制，算法。
urdf文件
	机器人各个组件之间的关键关系，用于生成静态的tf树，加入全局的tf关系中
maps文件
	其中pgm文件是用slam建出来的栅格地图，用0-255之间的数字表示占据概率。yaml文件用于导入pgm文件，并规定地图的分辨率以及地图坐标系的原点。
launch文件
	用于一次性导入启动
param文件
	包括硬件的参数和算法的参数。硬件参数可以更改串口的波特率，端口号。
	软件参数，包括机器人的速度，行进方式，避障的保守程度。

服务器启动方式：跑launch文件中的web_server.launch

src/process_switch.py：进程切换节点

launch/nav___.launch：保存小车当前位置角度，下次启动写入配置文件
